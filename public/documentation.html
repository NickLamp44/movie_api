MyFlix Movie API

The MyFlix API is built with the REST architecture in mind. This API has self-explanatory URLs, returns JSON responses, and uses standard HTTP response codes.

Request URL HTTP Method Request body data format Response data format
Get list a list of Users /users GET None An array of JSON objects holding data about each user
Gets data about a specific user /users/:name GET None A JSON object with data about a specific user
Adds a new user /users POST A JSON object holding data about the user to add Adds a new User to the database
Deletes Users /users/:name DELETE None Removes a User from the database
Updates a Users Favorite movie /users/:name/:favMovie PUT a user and a new movie updates a Users favorite movie in the data Base
Delete favMovie /users/:favMovie DELETE Removes a users favorite movie
List of Movies /movies GET None Returns an array of Movies
Gets data about a specific movie /movies/:title GET None Returns a JSON object containing the information about a specific user
Gets information about a specific director /movies/directors/:directorName GET None Returns data about a specific director
